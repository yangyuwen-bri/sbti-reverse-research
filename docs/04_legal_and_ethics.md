# Legal & Ethics Notes

## This repository does

- Reverse-engineer publicly delivered behavior for research/education.
- Provide independently written analysis code and docs.

## This repository does NOT

- Re-distribute full upstream frontend project as a clone.
- Re-host original assets (images, branding packs) in bulk.
- Claim official partnership or authorship.
- Provide medical/clinical psychological diagnosis.

## Recommended practice before publishing

1. Keep clear non-official disclaimer in README.
2. Credit original test URL and author name if publicly shown on page.
3. Avoid copying long original marketing copy/creative text blocks.
4. If rights owner requests specific takedown for copyrighted material,
   respond promptly and in good faith.

## User safety

- Present outputs as entertainment/system behavior analysis.
- Avoid labeling users with deterministic life conclusions.
- Encourage critical interpretation.
